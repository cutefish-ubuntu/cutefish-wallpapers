# cutefish-wallpapers

Cutefish System Wallpaper. 

## Build and Install

```sh
git clone https://github.com/cutefishos/wallpapers.git
mkdir ~/wallpapers/build && cd ~/wallpapers/build
cmake ..
make
sudo make install
```
